# my-codex-skills

個人用の Codex スキル置き場。

## 使い方
使いたいスキルのフォルダを、Codex のスキルディレクトリにコピーしてください。

```
~/.codex/skills/
```

コピー後、必要に応じて Codex を再起動（またはスキル再読み込み）します。
プロジェクトに合わせて SKILL.md を自由にカスタムしてください。

## 収録スキル

### db-design-owner
- 役割: DB設計の責任者として、要件から 2〜3 案の設計を提示
- 出力: 主要テーブル一覧、カラム/制約（PK/FK/unique/null）、主なインデックス、ER説明、設計の背景
- 特徴: マルチテナント前提（必要なら単一テナントに簡略化）、3NF寄りの設計、NULL最小化

### data-modeling-coach
- 役割: 初心者向けにデータモデリングを一緒に進めるコーチ
- 進め方: 概念モデル → 論理モデル → 物理モデルの順に対話で整理
- 出力: 最終的に ER説明、テーブル案、DDL を提示
